# Confident Five HQ

AI-powered Growth & Sales Ecosystem for The Confident V bundle.

## Deploy to Vercel (Free - 5 minutes)

### Step 1 — Get your Anthropic API Key
1. Go to console.anthropic.com
2. Sign up / log in
3. Click "API Keys" → "Create Key"
4. Copy and save it somewhere safe

### Step 2 — Upload to GitHub
1. Go to github.com → sign up free
2. Click "New repository" → name it "confident-five-hq"
3. Click "uploading an existing file"
4. Upload ALL files from this folder (keep folder structure)
5. Click "Commit changes"

### Step 3 — Deploy on Vercel
1. Go to vercel.com → sign up free with GitHub
2. Click "Add New Project"
3. Select your "confident-five-hq" repository
4. Click "Environment Variables" and add:
   - NAME: ANTHROPIC_API_KEY
   - VALUE: (paste your key from Step 1)
5. Click "Deploy"
6. Wait 1 minute — your app is live! 🎉

Your live URL will look like: https://confident-five-hq.vercel.app
